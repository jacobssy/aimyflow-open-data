## 主要スキル

- **インフラ設計**: 業務基盤となる拡張性あるインフラを設計する。
- **運用信頼性管理**: 監視、容量計画、パッチ、バックアップ、障害対応で安定稼働を守る。
- **クラウド・DC管理**: クラウドとオンプレミスをまたぐハイブリッド基盤を安定運用する。
- **セキュリティ統制**: アクセス制御、ハードニング、ログ、監査統制で要件に適合させる。
- **災害復旧計画**: 重要サービスを迅速に復旧するための戦略、切替手順、訓練計画を定める。
