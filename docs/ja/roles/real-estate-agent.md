## 主要スキル

- **物件査定**: 相場や立地、状態から適正価格を査定する。
- **販売戦略**: 価格設定や訴求内容、公開時期を設計して購入希望者を集める。
- **顧客適格確認**: 内見前に予算、資金計画、希望条件、購入時期を見極める。
- **条件交渉**: 価格、特約、修繕、引渡条件を調整して合意形成を進める。
- **取引実務管理**: 重要事項や契約書、期限、必要書類を決済まで正確に管理する。
