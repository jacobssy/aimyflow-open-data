## 主要スキル

- **オフィス運営管理**: 日々の管理業務の流れと対応状況を統括する。
- **予算・経費管理**: 管理部門の予算を立て、支出と運営コストを管理する。
- **規程・コンプライアンス管理**: 管理規程や文書基準を整備し、内部・法令要件への適合を保つ。
- **委託先・施設管理**: 委託先、契約、保守、職場サービスを調整し、施設運営を安定させる。
- **記録・業務プロセス統制**: 文書保存、承認フロー、事務手順を統制し、正確で監査可能な運用を保つ。
