## 主要スキル

- **契約交渉**: クラブやスポンサーと契約条件、報酬、肖像権を詰める。
- **選手代理**: 移籍交渉や進路判断、チームやリーグとの実務を代行する。
- **スポンサー契約**: 選手の価値や活動計画に合うスポンサー契約を組成する。
- **キャリア設計**: 競技寿命、移籍、収益、引退後を見据えて長期計画を立てる。
- **競技規程対応**: 登録、代理人規則、移籍期間、開示義務への対応を管理する。
