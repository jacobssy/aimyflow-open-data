- **メッセージ戦略**: 顧客理解とブランド方針から訴求の核を設計する。
- **見出し作成**: 注意を引き、訴求を瞬時に伝える見出しを書く。
- **媒体別調整**: 媒体特性や導線段階に合わせてコピーを調整する。
- **ブランドトーン管理**: 商品や対象に応じつつ、一貫したブランドトーンを保つ。
- **コピー校正**: 明確さ、表現の正確性、法務面を確認して公開品質に整える。
