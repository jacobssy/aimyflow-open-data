## 主要スキル

- **スケジュール調整**: 予定の重複を調整し、優先順に会議を整える。
- **文書作成管理**: 文書や報告書、社内記録を正確に作成・更新する。
- **出張・会議手配**: 出張、会議室、資料、行程を手配し、運営を円滑にする。
- **記録・ファイリング管理**: 保管ルールに沿って記録を整理し、必要時に迅速に取り出す。
- **経費・購買補助**: 経費を管理し、購買申請を処理し、社内規程との整合を確認する。
