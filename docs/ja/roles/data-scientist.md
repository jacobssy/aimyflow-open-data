## 主要スキル

- **データ前処理**: 分析やモデル化に使えるよう、生データを整形・結合・検証する。
- **統計解析**: 統計手法でデータの傾向、不確実性、関係性を定量化する。
- **予測モデリング**: 性能を測定しながら、結果を予測する機械学習モデルを構築・調整する。
- **モデル評価**: 適切な指標、検証設計、誤差分析で導入前のモデル品質を評価する。
- **実験設計**: 信頼できるプロダクトや事業判断につながるA/Bテストや観察研究を設計する。
