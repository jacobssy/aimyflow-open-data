## 主要スキル

- **システム設計**: 製品要件と運用制約に沿って拡張可能な構成とデータ流れを設計する。
- **実装技術**: 明確な構造と適切な抽象化で保守しやすく安定したコードを実装する。
- **テスト品質保証**: テスト設計と変更レビューで本番前に不具合や回帰を防ぐ。
- **性能最適化**: 実運用の負荷を分析し、遅延・処理量・資源効率を改善する。
- **デプロイ運用**: ビルドパイプライン、本番リリース、監視、障害対応を通じて安全に運用する。
