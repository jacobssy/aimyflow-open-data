## 主要スキル

- **データパイプライン構築**: 本番環境で動くバッチ・ストリーミング処理基盤を構築する。
- **データモデリング**: 分析精度と再利用性、性能を支えるデータモデルを設計する。
- **オーケストレーション運用**: 監視、リトライ、依存制御を備えたワークフローを安定運用する。
- **データ品質管理**: テストや検証ルール、監視でデータの信頼性を継続的に保つ。
- **データガバナンスとセキュリティ**: リネージ、アクセス制御、保持ルールを整備し、法令順守に対応する。
