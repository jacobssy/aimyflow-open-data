## 主要スキル

- **イメージ診断**: 顔立ちや体型、目的を見て似合う方向性を定める。
- **カラー診断**: 肌・髪・瞳との調和から似合う色調を見極める。
- **スタイリング設計**: 日常・仕事・特別な場面に合わせて服装や身だしなみを組み立てる。
- **ワードローブ整理**: 手持ち服を見直し、不要品や不足を整理して着回しを高める。
- **製品選定**: 要望、予算、手入れのしやすさに合わせて製品を選ぶ。
