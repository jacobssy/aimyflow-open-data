## 主要スキル

- **プログラム設計**: 地域課題と使命目標に沿って支援内容と実施計画を設計する。
- **助成金・予算管理**: 助成金条件、予算、支出基準を管理し、事業の適正運営を保つ。
- **成果測定**: 指標を定めて事業データを収集し、報告と改善のために成果を評価する。
- **連携先調整**: 地域団体、委託先、紹介機関を調整し、各拠点で安定して支援を提供する。
- **安全管理とコンプライアンス**: 利用者保護、個人情報、報告義務の要件を運用し、参加者の安全を守る。
