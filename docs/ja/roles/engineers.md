## 主要スキル

- **技術設計**: 要件を実現可能な設計仕様へ落とし込む。
- **解析・シミュレーション**: 計算やシミュレーションで性能を予測し設計を検証する。
- **試験・検証**: 機能・安全・信頼性の目標を満たすか試験で確認する。
- **規格適合**: 設計・製造・出荷の各段階で規格や法規を適用する。
- **原因解析**: 現場や生産での不具合を解析し是正設計につなげる。
