## 主要スキル

- **レファレンス面談**: 利用者の調査意図を整理し、適切な検索方針に落とし込む。
- **学術データベース検索**: 分野別データベースや目録、索引を使い、信頼できる学術情報を効率よく探す。
- **資料評価**: 研究支援に向けて、資料の関連性・権威性・学術的妥当性を見極める。
- **情報リテラシー指導**: 検索・引用・情報の適正利用について、授業と連動した指導を行う。
- **学術引用支援**: 引用スタイルや文献管理ツール、剽窃防止について利用者を支援する。
