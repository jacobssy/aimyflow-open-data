## 主要スキル

- **予算計画**: 事業目標と制約に沿って年度予算とローリング計画を策定する。
- **業績予測**: 売上・費用・資金・人員の前提を更新し月次予測を作成する。
- **予実差異分析**: 実績と計画の差異を説明し業績ギャップの要因を特定する。
- **経営報告**: 月次締めやレビュー、経営会議向けにKPI報告と意思決定材料を提供する。
- **事業採算モデリング**: シナリオモデル、投資収益性、リスク前提を用いて投資案件を評価する。
