## Core Skills

- **System Monitoring**: Track system health, logs, and alerts to detect abnormal conditions in live environments.
- **Incident Response**: Diagnose outages, restore service quickly, and escalate issues using operational procedures.
- **Job Scheduling**: Run and control batch jobs, file transfers, and scheduled tasks to keep operations on time.
- **Change Execution**: Apply approved changes in production with checklists, timing control, and rollback readiness.
- **Runbook Compliance**: Follow runbooks, record operational actions, and maintain audit-ready evidence of system handling.
