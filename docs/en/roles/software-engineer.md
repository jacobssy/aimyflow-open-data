## Core Skills

- **System Design**: Designs scalable services, data flows, and interfaces that fit product and operational constraints.
- **Code Implementation**: Builds maintainable application code with clear structure, solid abstractions, and reliable behavior.
- **Testing & Quality**: Defines tests, reviews changes, and catches regressions before software reaches production.
- **Performance Optimization**: Profiles bottlenecks and improves latency, throughput, and resource usage in real workloads.
- **Deployment & Operations**: Ships software safely through build pipelines, production releases, monitoring, and incident handling.
