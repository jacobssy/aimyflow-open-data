## Core Skills

- **Reference Interviewing**: Clarifies a patron’s research need and translates it into an effective search approach.
- **Academic Database Searching**: Uses subject databases, catalogs, and indexes to locate authoritative scholarly sources efficiently.
- **Source Evaluation**: Assesses relevance, authority, and scholarly quality when selecting materials for research support.
- **Information Literacy Instruction**: Designs and delivers course-integrated instruction on searching, citing, and using information ethically.
- **Scholarly Citation Support**: Guides users on citation styles, reference tools, and avoiding plagiarism in academic work.
