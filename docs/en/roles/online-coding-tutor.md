## Core Skills

- **Curriculum Scaffolding**: Structures coding lessons into progressive steps that match the learner’s level and goals.
- **Live Code Debugging**: Diagnoses learner code in real time and explains fixes without taking over the solution.
- **Code Review Feedback**: Reviews assignments for correctness, readability, and technique, then gives actionable improvement notes.
- **Concept Breakdown**: Turns abstract programming concepts into clear examples, exercises, and mental models learners can use.
- **Learning Progress Assessment**: Assesses skill growth through exercises and session performance, then adjusts the teaching plan accordingly.
