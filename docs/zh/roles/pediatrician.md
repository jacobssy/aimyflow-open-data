# 儿科医生

## 概览

- 匹配工具数: 4
- 技能卡片数: 5

## 核心技能

- **儿科评估**: 评估生长发育与症状
- **疫苗管理**: 制定接种计划并随访
- **儿童急症处理**: 处理儿童常见急症
- **慢病随诊**: 随访哮喘糖尿病等
- **发育筛查**: 筛查发育与行为问题

## 匹配工具

- [bolt-new](https://www.aimyflow.com/zh/ai/bolt-new): Bolt 是 AI 应用构建平台，可将提示词快速生成可用的 Web 应用，适合创始人、产品团队和原型开发者。在 AI 时代，它能加速从想法到原型的验证流程。
- [dorascribe-ai](https://www.aimyflow.com/zh/ai/dorascribe-ai): DoraScribe 是一款 AI 医疗书记员与转录工具，可将患者问诊内容转换为 SOAP 病历记录及其他图表文档，供医生和其他临床人员使用，并提供移动端访问和符合 HIPAA 标准的数据处理。对于医生和临床团队而言，它能够减轻文档记录负担，支持生成更准确、一致的病历，同时让注意力持续集中在患者护理上。
- [openrouter-ai](https://www.aimyflow.com/zh/ai/openrouter-ai): OpenRouter 是一个面向大语言模型的统一 API 与路由平台，帮助开发者通过单一接口接入、比较和使用多个模型，让团队更灵活优化成本、性能与稳定性。
- [radily-ai](https://www.aimyflow.com/zh/ai/radily-ai): Radily 是一款符合 HIPAA 要求的 AI 工具，可用通俗易懂的英语解释 CT、MRI 和 X 光报告，并提供可选的视频摘要和建议的后续提问，主要面向希望安全地理解其影像检查结果的患者。 在 AI 赋能的医疗沟通中，它可帮助放射科医生和转诊临床医生通过在临床讨论前后为患者提供更清晰的报告解读，减少患者的困惑。
