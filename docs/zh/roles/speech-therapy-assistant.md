# 言语治疗助理

[在 AimyFlow 打开完整职业页](https://www.aimyflow.com/zh/role/speech-therapy-assistant)

## 概览

- 匹配工具数: 2
- 技能卡片数: 5

## 核心技能

- **治疗协助**: 备齐教具并协助训练
- **进展观察**: 记录反应与训练表现
- **居家练习指导**: 指导患者居家练习
- **治疗记录**: 规范整理治疗记录
- **安全与设备准备**: 按需准备安全器材

## 匹配工具

- [bolt-new](https://www.aimyflow.com/zh/ai/bolt-new): Bolt 是 AI 应用构建平台，可将提示词快速生成可用的 Web 应用，适合创始人、产品团队和原型开发者。在 AI 时代，它能加速从想法到原型的验证流程。
- [openrouter-ai](https://www.aimyflow.com/zh/ai/openrouter-ai): OpenRouter 是一个面向大语言模型的统一 API 与路由平台，帮助开发者通过单一接口接入、比较和使用多个模型，让团队更灵活优化成本、性能与稳定性。
