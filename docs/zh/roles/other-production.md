## 核心技能

- **生产操作**: 按标准完成生产操作
- **设备调机**: 按批次完成设备切换
- **过程质检**: 生产中监控质量偏差
- **生产记录**: 准确记录批次参数
- **安全合规**: 遵守安全与现场规范

## 匹配工具

- [openrouter-ai](https://www.aimyflow.com/zh/ai/openrouter-ai): OpenRouter 是一个面向大语言模型的统一 API 与路由平台，帮助开发者通过单一接口接入、比较和使用多个模型，让团队更灵活优化成本、性能与稳定性。
- [replit-com](https://www.aimyflow.com/zh/ai/replit-com): Replit 是一款 AI 驱动的开发平台，帮助用户在浏览器中构建应用和网站，主要面向开发者、创业者和快速交付软件的团队。它缩短了从想法到可运行原型的路径，帮助构建者以更少配置快速迭代和部署。
