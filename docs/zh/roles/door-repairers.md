# 门维修工

[在 AimyFlow 打开完整职业页](https://www.aimyflow.com/zh/role/Door-Repairers)

## 概览

- 匹配工具数: 1
- 技能卡片数: 5

## 核心技能

- **故障诊断**: 判断卡滞下垂等故障原因
- **合页校正**: 调整合页门框恢复顺畅开关
- **锁具维修**: 修复更换锁具闭门器等配件
- **门板修补**: 修补门板裂损变形磨损
- **防火安全校验**: 确保维修后符合防火安全要求
