## Habilidades clave

- **Análisis químico**: Realiza análisis con métodos validados para determinar composición, pureza e impurezas traza.
- **Desarrollo de métodos**: Diseña y optimiza métodos de laboratorio para obtener resultados fiables, reproducibles y adecuados al uso.
- **Diseño de reacciones**: Selecciona reactivos, condiciones y rutas para obtener compuestos objetivo con rendimiento y selectividad controlados.
- **Interpretación de datos**: Interpreta espectros, cromatogramas y resultados experimentales para confirmar estructuras y explicar comportamientos observados.
- **Cumplimiento de seguridad**: Aplica normas de seguridad química, gestión de residuos y documentación para mantener el trabajo de laboratorio conforme y controlado.
