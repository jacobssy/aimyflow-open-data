## Habilidades clave

- **Diseño de programas**: Diseña modelos de intervención, actividades y cronogramas alineados con necesidades comunitarias y misión.
- **Gestión de subvenciones y presupuesto**: Controla fondos restringidos, presupuestos y reglas de gasto para sostener programas viables y en regla.
- **Medición de resultados**: Define indicadores, recopila datos del programa y evalúa resultados para reportar y mejorar.
- **Coordinación de socios**: Coordina socios comunitarios, proveedores y redes de derivación para prestar servicios de forma consistente.
- **Protección y cumplimiento**: Aplica requisitos de protección, privacidad y reporte para resguardar a participantes y cumplir la normativa.
