## Habilidades clave

- **Estrategia de medios**: Diseña planes de canales alineando mensajes, audiencias, tiempos y objetivos de campaña.
- **Relación con prensa**: Gestiona el contacto con periodistas, el pitching, el seguimiento y el envío de materiales.
- **Desarrollo de mensajes**: Define narrativas, mensajes clave y argumentarios para cada audiencia y medio.
- **Producción de contenidos**: Produce notas de prensa, dosieres, comunicados y contenidos digitales listos para difusión.
- **Monitoreo de medios**: Sigue cobertura, tono, alcance y señales de riesgo para medir impacto y orientar la respuesta.
