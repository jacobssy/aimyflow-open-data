## Habilidades clave

- **Diseño de sistemas**: Diseña servicios, flujos de datos e interfaces escalables según las restricciones del producto y la operación.
- **Implementación de código**: Desarrolla código mantenible con estructura clara, buenas abstracciones y comportamiento confiable.
- **Pruebas y calidad**: Define pruebas, revisa cambios y detecta regresiones antes de que el software llegue a producción.
- **Optimización de rendimiento**: Analiza cuellos de botella y mejora latencia, rendimiento y uso de recursos en cargas reales.
- **Despliegue y operación**: Entrega software con seguridad mediante pipelines, despliegues, monitoreo y respuesta a incidentes.
