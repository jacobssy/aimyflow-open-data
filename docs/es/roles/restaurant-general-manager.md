## Habilidades clave

- **Gestión de pérdidas y ganancias**: Controla ventas, mano de obra y costos operativos para alcanzar la rentabilidad objetivo.
- **Operación del servicio**: Planifica cobertura, ritmo y ejecución de turnos para mantener un servicio fluido en horas punta.
- **Cumplimiento sanitario**: Aplica normas de higiene, almacenamiento y auditoría para reducir riesgos sanitarios y regulatorios.
- **Control de inventario**: Gestiona compras, niveles de stock y mermas para asegurar disponibilidad, frescura y margen.
- **Recuperación del cliente**: Resuelve fallos de servicio con rapidez y criterio para recuperar la satisfacción y la reputación.
