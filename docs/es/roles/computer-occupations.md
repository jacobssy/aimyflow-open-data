## Habilidades clave

- **Desarrollo de software**: Diseña, desarrolla, prueba y mantiene aplicaciones para usos empresariales o técnicos.
- **Administración de sistemas**: Administra servidores, entornos de usuario e infraestructura para asegurar estabilidad y disponibilidad.
- **Gestión de bases de datos**: Estructura, protege y optimiza plataformas de datos para un almacenamiento fiable y acceso rápido.
- **Operaciones de ciberseguridad**: Supervisa amenazas, aplica controles y responde a incidentes para proteger sistemas y datos.
- **Soporte técnico**: Diagnostica incidencias de usuarios y sistemas, restablece el servicio y documenta soluciones recurrentes.
