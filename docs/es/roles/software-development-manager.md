## Habilidades clave

- **Hoja de ruta técnica**: Convierte objetivos de producto en fases de ingeniería, dependencias e hitos de entrega.
- **Gobierno de arquitectura**: Orienta decisiones de diseño para mantener sistemas escalables, mantenibles y alineados entre equipos.
- **Gestión de entregas**: Gestiona alcance, capacidad y riesgos de ejecución para entregar versiones confiables a tiempo.
- **Operaciones de ingeniería**: Mejora el flujo de desarrollo con métricas, herramientas, prácticas de CI/CD y retroalimentación de incidentes.
- **Supervisión de seguridad y cumplimiento**: Garantiza que las prácticas de entrega cumplan requisitos de seguridad, privacidad y regulación durante todo el ciclo.
