## Habilidades clave

- **Levantamiento de linderos**: Determina los límites legales de una propiedad con documentos y evidencia de campo.
- **Medición geodésica**: Utiliza GNSS, estación total y redes de control para obtener coordenadas y cotas precisas.
- **Cartografía topográfica**: Levanta terreno, elementos y servicios para generar planos fiables de diseño y planificación.
- **Replanteo de obra**: Traslada las coordenadas del proyecto al terreno para ubicar correctamente las obras.
- **Ajuste de datos topográficos**: Verifica la calidad, ajusta observaciones y documenta resultados conforme a norma.
