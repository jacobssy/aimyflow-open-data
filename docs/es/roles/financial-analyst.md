## Habilidades clave

- **Modelización financiera**: Construye modelos integrados para proyectar resultados, flujo de caja y efectos de financiamiento.
- **Análisis de valoración**: Evalúa el valor de empresas o activos mediante múltiplos, DCF y transacciones comparables.
- **Presupuestación y previsión**: Elabora presupuestos y previsiones continuas vinculando impulsores operativos con resultados financieros.
- **Análisis de desviaciones**: Identifica y explica diferencias entre real, presupuesto y periodos previos para apoyar decisiones.
- **Reporting financiero**: Prepara informes de gestión claros con KPI, tendencias e indicadores de riesgo para las partes interesadas.
