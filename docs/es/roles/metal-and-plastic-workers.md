## Habilidades clave

- **Preparación de máquinas**: Ajusta prensas, cortadoras, moldeadoras o extrusoras según el material, las dimensiones y las tolerancias.
- **Procesado de materiales**: Corta, dobla, moldea, recorta o acaba piezas metálicas y plásticas con el proceso adecuado para cada material.
- **Inspección dimensional**: Mide piezas terminadas con calibres e instrumentos para verificar dimensiones, ajuste y calidad superficial.
- **Seguimiento de producción**: Supervisa la producción, ajusta avances o temperaturas y corrige desvíos para mantener un resultado estable.
- **Control de utillaje y seguridad**: Revisa utillajes, resguardos y puestos de trabajo para evitar daños, operar con seguridad y reducir paradas.
