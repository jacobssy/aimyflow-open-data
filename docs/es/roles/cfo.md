## Habilidades clave

- **Planificación financiera**: Elabora presupuestos, previsiones y planes financieros para orientar el rendimiento.
- **Gestión de tesorería**: Controla liquidez, necesidades de financiación y asignación de capital.
- **Información financiera**: Supervisa estados financieros, cierres contables e informes para dirección.
- **Riesgo y cumplimiento**: Mantiene controles, cumplimiento normativo y supervisión del riesgo financiero.
- **Estrategia de capital**: Evalúa financiación, inversiones y M&A para sostener el valor empresarial.
