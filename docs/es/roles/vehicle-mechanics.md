## Habilidades clave

- **Diagnóstico de fallas**: Identifica fallas mecánicas y electrónicas con inspecciones, equipos de prueba y datos del sistema.
- **Reparación de motor y transmisión**: Repara y sustituye componentes del motor, la transmisión y la cadena cinemática para recuperar el rendimiento.
- **Servicio de frenos y suspensión**: Inspecciona y mantiene los sistemas de frenos, dirección y suspensión para una circulación segura.
- **Servicio del sistema eléctrico**: Prueba y repara baterías, cableado, sensores, iluminación y sistemas de carga o arranque.
- **Mantenimiento preventivo**: Realiza servicios programados, cambios de fluidos y controles de desgaste para evitar averías y prolongar la vida útil.
