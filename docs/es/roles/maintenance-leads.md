## Habilidades clave

- **Planificación de mantenimiento preventivo**: Define y ajusta planes de mantenimiento para reducir fallas y paradas.
- **Diagnóstico de fallas**: Dirige el análisis técnico para hallar la causa raíz y restablecer el equipo con seguridad.
- **Gestión de órdenes de trabajo**: Prioriza, asigna y da seguimiento a las órdenes para mantener la ejecución bajo control.
- **Control de seguridad y cumplimiento**: Aplica bloqueos, permisos y estándares de mantenimiento para cumplir seguridad y normativa.
- **Gestión de repuestos**: Controla repuestos críticos, niveles de stock y reposición para asegurar el mantenimiento.
