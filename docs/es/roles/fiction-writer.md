## Habilidades clave

- **Arquitectura narrativa**: Diseña trama, ritmo y giros en una narración extensa y coherente.
- **Construcción de personajes**: Construye personajes distintivos con motivaciones, voz y evolución creíbles.
- **Voz narrativa**: Controla estilo, punto de vista y tono para sostener la experiencia de lectura.
- **Construcción de mundos**: Define escenarios, reglas y contexto para sostener la historia con coherencia.
- **Revisión de manuscritos**: Reelabora borradores para afinar estructura, prosa, continuidad e impacto de escenas.
