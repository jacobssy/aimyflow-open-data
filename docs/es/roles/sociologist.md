## Habilidades clave

- **Diseño de investigación**: Diseña estudios sociológicos con preguntas, muestras, métodos y lógica analítica claras.
- **Métodos de encuesta y entrevista**: Diseña instrumentos y realiza encuestas o entrevistas para obtener datos sociales fiables.
- **Análisis cualitativo**: Interpreta relatos, observaciones y documentos para identificar patrones y significados sociales.
- **Análisis estadístico**: Aplica técnicas estadísticas para probar relaciones, comparar grupos y evaluar tendencias sociales.
- **Ética de la investigación**: Protege a los participantes mediante consentimiento, confidencialidad y manejo cuidadoso de datos sensibles.
