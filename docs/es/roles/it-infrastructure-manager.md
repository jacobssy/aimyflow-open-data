## Habilidades clave

- **Arquitectura de infraestructura**: Diseña bases escalables de cómputo, almacenamiento, red y plataforma para los servicios del negocio.
- **Confiabilidad operativa**: Mantiene la disponibilidad mediante monitoreo, planificación de capacidad, parches, copias e incidentes.
- **Gestión de nube y centro de datos**: Opera infraestructura híbrida entre nube y entorno local con niveles de servicio consistentes.
- **Controles de seguridad y cumplimiento**: Implementa accesos, hardening, registros y controles de auditoría para cumplir requisitos normativos.
- **Planificación de recuperación ante desastres**: Define estrategias de recuperación, procedimientos de conmutación y pruebas para restaurar servicios críticos.
