## Habilidades clave

- **Diseño observacional**: Planifica observaciones con telescopio ajustando objetivos, instrumentos, tiempo de exposición y restricciones al fin científico.
- **Reducción de datos**: Convierte datos brutos del detector en espectros, imágenes o series temporales calibradas controlando ruido y artefactos instrumentales.
- **Modelado físico**: Construye y contrasta modelos físicos de estrellas, galaxias, plasmas o sistemas cosmológicos con observaciones.
- **Inferencia estadística**: Estima parámetros, incertidumbres y significancia de detección con métodos estadísticos rigurosos para datos escasos o ruidosos.
- **Computación científica**: Desarrolla código reproducible y flujos de simulación para procesar grandes volúmenes de datos y comparar teoría con mediciones.
