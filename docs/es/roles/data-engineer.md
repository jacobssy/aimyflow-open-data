## Resumen

- Herramientas asociadas: 85
- Tarjetas de habilidades: 5

## Habilidades clave

- **Ingeniería de pipelines**: Diseña pipelines batch y streaming fiables para mover y transformar datos a escala de producción.
- **Modelado de datos**: Diseña modelos para warehouse y lakehouse que mejoran precisión analítica, reutilización y rendimiento.
- **Orquestación y operación**: Opera flujos programados con monitoreo, reintentos, control de dependencias y respuesta a incidentes.
- **Gestión de calidad de datos**: Define pruebas, reglas de validación y controles de observabilidad para mantener datos confiables.
- **Gobierno y seguridad de datos**: Implementa linaje, controles de acceso y reglas de retención para cumplir requisitos normativos y de protección.
