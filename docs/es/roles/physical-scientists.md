## Habilidades clave

- **Diseño experimental**: Diseña estudios controlados para probar hipótesis físicas con variables medibles y comparaciones válidas.
- **Instrumentación y medición**: Opera, calibra y valida instrumentos científicos para obtener datos precisos y trazables.
- **Análisis y modelado de datos**: Analiza resultados experimentales con modelos estadísticos y físicos para cuantificar patrones, incertidumbre y ajuste.
- **Validación teórica**: Evalúa si las observaciones respaldan, ajustan o cuestionan teorías y supuestos físicos existentes.
- **Seguridad y cumplimiento en laboratorio**: Aplica protocolos de seguridad, control de materiales peligrosos y estándares de documentación durante la investigación.
