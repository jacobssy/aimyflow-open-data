## Habilidades clave

- **Modelado cuantitativo**: Construye modelos matemáticos para representar sistemas, probar supuestos y obtener predicciones útiles.
- **Diseño experimental**: Planifica experimentos con variables controladas, metodología sólida y resultados medibles.
- **Análisis de datos**: Procesa e interpreta datos científicos o numéricos para identificar patrones, incertidumbre y significancia.
- **Cálculo científico**: Utiliza herramientas computacionales para resolver ecuaciones, simular procesos y automatizar análisis.
- **Informe científico**: Documenta métodos, resultados y limitaciones en un formato reproducible para revisión o aplicación.
