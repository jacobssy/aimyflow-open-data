# GitHub Repo Setup

Use the following values when creating the public GitHub repository.

## Repository Name

Recommended:

- `awesome-aitools`

## Website

```text
https://www.aimyflow.com
```

## Description

Primary option:

```text
A curated multilingual list of AI tools, with highlighted picks on GitHub and full directory links back to AimyFlow.
```

Alternative option:

```text
Awesome-style AI tools list curated from the AimyFlow directory, with multilingual docs and structured exports.
```

## Launch Checklist

- Set the repository to public
- Set the repository Website to `https://www.aimyflow.com`
- Paste the Description above into the About box
- Add the recommended Topics
- Push the generated `data/` and `docs/` files
- Enable GitHub Actions
- Add the required repository secrets
- Optionally enable GitHub Pages for the `/docs` experience
